This folder is dedicated to the explanation of linear regression theory. Not all topics have been addressed, but an important part of them has been covered. Here you can find the following fields:

CSV
* winequality-red.csv --> Database used. You can also download from: https://www.kaggle.com/datasets/uciml/red-wine-quality-cortez-et-al-2009

HTML
* linear-regression-theory-and-python.html --> Html with the explanations and the code. The output of the code is hidden under a button in each cell to make the reading easier.

PYTHON
* linear-regression-theory-and-python.py --> Python script with the same information as the html.
* requirements.txt --> Field with the packages used in the script.
